NERF-Wars
=========

NERF wars sessions manager: random start, ending coutdown, napalm, music, helicopters.

## Installation

```bash
npm install
```

## Tests

using `mocha`:

```bash
npm install -g mocha
mocha
```

## Code coverage

using `instanbul`

```bash
npm install -g istanbul
istanbul cover _mocha -- -R spec
open coverage/lcov-report/index.html
```
